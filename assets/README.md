# Assets

This folder holds **system architecture** sources and optional **screenshots** for documentation and portfolio use.

| File | Purpose |
|------|---------|
| `system-architecture.mmd` | Mermaid source for the end-to-end flow (input → agent → guardrails → output → verification). |

To produce a PNG for GitHub or slides:

1. Open [Mermaid Live Editor](https://mermaid.live) and paste the contents of `system-architecture.mmd`.
2. Export as PNG and save here (e.g. `system-architecture.png`).

The same diagram is embedded in the root `README.md` under **Architecture** so it renders on GitHub without an extra image file.
