# Assets

All **diagrams** and **screenshots** for this repo live here so the project root stays clean and graders can find media in one place.

| File | Purpose |
|------|---------|
| `demo.png` | Streamlit UI screenshot referenced from the root `README.md`. |
| `system-architecture.mmd` | Mermaid source for the **system** flow (input → agent → guardrails → output). |
| `system-architecture.png` | Exported system diagram (same graph as in README / Mermaid block). |
| `class-diagram.mmd` | Mermaid **class** diagram source (`Owner`, `Pet`, `Task`, `Scheduler`). |
| `uml-class-diagram.png` | Exported class UML image for slides or docs. |

To refresh PNG exports:

1. Open [Mermaid Live Editor](https://mermaid.live) and paste the `.mmd` source.
2. Export as PNG and replace the matching file in this folder.
